### DiffOperation 类型

```
┌─────────────────────────────────────────────────────────────────────────────┐
│              DiffOperation 类型                                    │
│                                                                  │
│  update_line:    │  更新单行 (从起始列到结束)                │
│  delete_lines:   │  删除多行 (n 行)                             │
```
